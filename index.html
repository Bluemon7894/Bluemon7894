1. 라즈베리파이 5 우분투 설치

1.1 라즈베리파이 OS 설치 준비

이 섹션에서는 라즈베리파이 5에 우분투를 설치하는 방법을 설명합니다. 준비물로는 라즈베리파이 5 보드, 마이크로 SD 카드(최소 16GB), 전원 어댑터, 모니터, 키보드, 마우스가 필요합니다.

1.2 우분투 이미지 다운로드 및 SD 카드 준비

  1. 우분투 이미지를 다운로드: 우분투 공식 웹사이트에서 Raspberry Pi 용 우분투 이미지를 다운로드합니다. Raspberry Pi 5에 적합한 버전을 선택하세요.
  2. Raspberry Pi Imager 설치: Raspberry Pi Imager를 다운로드하고 설치합니다.
  3. SD 카드에 이미지 굽기: Raspberry Pi Imager를 실행한 후, 다운로드한 우분투 이미지를 선택하고 SD 카드에 굽습니다.

1.3 라즈베리파이 부팅 및 초기 설정

  1. SD 카드 삽입: 이미지를 굽는 것이 완료되면 SD 카드를 라즈베리파이에 삽입합니다.
  2. 전원 연결 및 부팅: 라즈베리파이에 전원을 연결하여 부팅합니다. 첫 부팅 시 우분투 설정 화면이 나타납니다.
  3. 네트워크 설정: Wi-Fi 또는 유선 인터넷을 통해 라즈베리파이를 네트워크에 연결합니다.
  4. 언어 및 시간대 설정: 언어 및 시간대를 설정하고, 필요한 경우 키보드 레이아웃도 선택합니다.
  5. 업데이트 수행: 터미널을 열어 최신 소프트웨어 업데이트를 수행합니다. 명령어는 다음과 같습니다:
    6. sudo apt update && sudo apt upgrade -y

2. 하드웨어 설정

2.1 하드웨어 부품 목록

  • 라즈베리파이 5
  • RGB LED 스트립 (5V용)
  • NPN 트랜지스터 3개 (예: 2N2222)
  • 330Ω 저항 3개
  • 브레드보드
  • 점퍼 케이블 (암/수)
  • 마이크로 USB 마이크 (사운드 입력용)

2.2 하드웨어 배선 연결

RGB LED 스트립과 트랜지스터를 라즈베리파이에 연결합니다. 아래는 각 연결 방법입니다:

  1. 라즈베리파이 GPIO 핀 설정: GPIO 17 (Red LED), GPIO 22 (Green LED), GPIO 27 (Blue LED) 핀을 사용합니다.
  2. 트랜지스터와 저항 연결: 330Ω 저항을 각 트랜지스터의 베이스(Base) 핀과 라즈베리파이의 해당 GPIO 핀 사이에 연결합니다. 트랜지스터의 컬렉터(Collector) 핀을 RGB LED 스트립의 각 색상 채널(R, G, B)에 연결하고, 이미터(Emitter) 핀을 GND에 연결합니다.
  3. RGB LED 스트립 연결: RGB LED 스트립의 VCC 핀을 라즈베리파이의 5V 핀에, GND 핀을 라즈베리파이의 GND 핀에 연결합니다.

3. 머신러닝 모델 준비 (TensorFlow Lite)

3.1 티처블 머신에서 모델 생성

티처블 머신을 사용하여 사운드 인식 모델을 생성합니다. 아래 단계에 따라 진행합니다:

  1. 티처블 머신 웹사이트 접속: Teachable Machine에 접속합니다.
  2. 프로젝트 생성: 'Audio Project'를 선택한 후 'Standard' 모델 타입을 선택합니다.
  3. 데이터 수집: 'Background Noise', 'Horn', 'Siren' 등의 클래스를 추가하고, 학습 데이터 수집을 위해 소리를 녹음하거나 업로드합니다.
  4. 모델 학습: 데이터 수집이 완료되면 'Train Model'을 클릭하여 모델을 학습시킵니다.
  5. 모델 내보내기: 학습이 완료된 모델을 'TensorFlow Lite' 형식으로 내보내고, 'Download My Model'을 클릭하여 `.tflite` 모델 파일과 레이블 파일(`.txt`)을 다운로드합니다.

3.2 모델 파일 라즈베리파이로 전송

다운로드한 `.tflite` 모델 파일과 레이블 파일을 SCP(Secure Copy Protocol) 등을 사용하여 라즈베리파이로 전송합니다:

scp soundclassifier_with_metadata.tflite pi@<라즈베리파이_IP>:/home/pi/
scp labels.txt pi@<라즈베리파이_IP>:/home/pi/

4. 코드 및 라이브러리 설치

4.1 필요한 파이썬 라이브러리 설치

라즈베리파이에서 사운드 인식과 LED 제어를 위해 필요한 라이브러리를 설치합니다:

  • pip 설치: sudo apt install python3-pip
  • 필요한 라이브러리 설치: pip3 install numpy tensorflow pyaudio gpiozero
  • tkinter 설치: sudo apt-get install python3-tk

4.2 파이썬 코드 실행

아래는 RGB LED 제어와 사운드 인식을 위한 최종 파이썬 코드입니다. 이 코드를 사용하여 모델이 올바르게 예측하고 LED를 제어할 수 있는지 확인합니다.

Python 스크립트를 실행하려면 터미널에서 다음 명령어를 입력합니다:

python3 sound_led_control.py

배경 소음, 경적 소리, 사이렌 소리 등 다양한 소리를 재생하여 모델이 올바르게 예측하고 LED를 제어하는지 테스트합니다.
